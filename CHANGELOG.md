## 0.1.0 — 2026-06-21
First public release.

- Capture any selection as a beautiful image, inline in the editor
- 9 hand-tuned themes (Midnight, GitHub Dark, Dracula, Tokyo Night, Nord, Synthwave, Aurora, Ember, Paper)
- Separate backdrop picker (8 gradients) + Transparent PNG export
- Font + size controls, adjustable padding, optional line numbers
- macOS-style window chrome (toggle)
- 🎲 Surprise button to randomize the look
- Copy to clipboard or Save PNG up to 4x
- 100% local — no account, no telemetry
